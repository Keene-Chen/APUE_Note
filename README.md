# APUE 阅读笔记
《UNIX环境高级编程》（英语：Advanced Programming in the UNIX Environment，简称APUE）是理查德·史蒂文斯（Richard Stevens）写的一本介绍UNIX环境C语言编程接口的书籍。
被UNIX C程序员誉为“圣经”。                               --Wikipedia

## 推荐阅读顺序

### I/O 操作
第三、五、十四章节

### 文件系统
第四、六、七章节

### 并发编程
第十、十一章节

其中第十章为信号，十一章为多线程

### 进程间通信（IPC）
第八、十三、十五、十六章节

其中第八章为进程基础，十三章为守护进程，十五章为本地IPC，十六为网络IPC（socket）
